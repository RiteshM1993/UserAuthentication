### Prerequisites


* pyenv https://github.com/pyenv/pyenv
* pipenv: `python3 -m pip install -Iv pipenv==11.8.0`



### Setup

   * `git clone https://github.com/tbikeev/BCM_multihosted`
   * `cd BCM_multihosted`
   * `pipenv install`
   * `make test`, sames as `pipenv run python ./manage.py test`
